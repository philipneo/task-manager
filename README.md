# Task Manager

A minimal task manager for organizing priorities, due dates, and daily work.

## Features

- Add tasks with priority and due date
- Mark tasks complete
- Filter by active or completed tasks
- Store tasks in localStorage
- Clean responsive layout

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Goals

This project practices DOM rendering, state management, event handling, and simple product thinking.
